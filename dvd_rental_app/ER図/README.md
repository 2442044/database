# ER図ドキュメント

このフォルダには SchemaSpy によって生成されたデータベース定義書が含まれています。

## 閲覧方法

### 1. 全体ER図を見る
データベース全体のテーブル関係図（ER図）は以下の画像ファイルで直接確認できます。
- **[全体ER図 (画像)](diagrams/summary/relationships.real.large.png)**

### 2. 詳細なドキュメントを見る (HTML)
詳細なレポート（テーブル定義、カラム情報など）は `index.html` から閲覧できますが、GitHub上では直接表示されません。

**方法 A: ダウンロードして閲覧**
このリポジトリをダウンロード（Clone または Download ZIP）し、ローカル環境で `index.html` をブラウザで開いてください。

**方法 B: GitHub Pages で公開**
リポジトリの "Settings" > "Pages" から GitHub Pages を有効にすると、ブラウザ上で閲覧できるようになります。
